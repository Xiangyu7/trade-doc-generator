# trade-doc-generator

外贸单据自动生成工具（RPA）。把客户 PO 拖进网页即可自动预填并生成
形式发票(PI) / 商业发票(CI) / 装箱单(PL)，并支持清关文件合并、危化品单据生成。

> 详细操作见 [`使用说明.txt`](使用说明.txt)。

## 功能

- **生成发票**：上传 PO（PDF / 图片，图片走本机 OCR）→ 自动抽取字段 → 人工核对 → 一键导出 PI/CI/PL（Excel + PDF）。
- **合并清关包**：把提单 / 保险单 / COA / 产地证等（PDF / Excel / Word / 图片）拖入，调序后合并成单个 PDF。
- **危化品文档**：从性能单 / 分类鉴定报告提取字段，生成符合性声明 / 责任声明 / 厂检单。

## 运行环境

- Python 3，依赖：`flask`、`openpyxl`、`pypdf`、`playwright`（仅 e2e 测试用）。
- LibreOffice（用于导出 PDF，需在 PATH 中）。
- macOS：图片 / 扫描件 OCR 由 `ocr_helper.swift` 首次运行时自动编译为 `ocr_helper`。

## 使用前需自备的文件（**不在仓库内**，含各公司专属数据）

为避免泄露真实银行账户 / 客户信息，以下文件已从仓库排除，请使用者放到对应位置：

| 文件 | 位置 | 说明 |
| --- | --- | --- |
| `.ai_key` | 本目录下 | MiniMax API key（明文一行）。不配也能用，会回退本地正则识别。 |
| `CPH模板.xlsx` | **上级目录** | 发票/箱单主模板，含你自己的卖方抬头与银行信息。 |
| 危化品 docx 模板 | **上级目录** | 符合性声明 / 责任声明 / 厂检单模板。 |
| `订单信息表.xlsx` | 本目录下 | 批量命令行版的订单输入表。 |

> 注意：`core.py` 里的危化品替换规则使用「模板内现有示例值」作为查找锚点，
> 换成你自己的模板后，需把这些锚点改成模板里对应的实际文字。

## 启动

```bash
python3 app.py        # 网页版，浏览器打开 http://127.0.0.1:5000
python3 生成单据.py    # 批量命令行版，结果在 输出单据/
```

## AI 配置（可选）

- key：`.ai_key` 文件，或环境变量 `MINIMAX_API_KEY`。
- 模型 / 接口：`MINIMAX_MODEL`、`MINIMAX_BASE_URL`（默认 MiniMax-M3 / api.minimaxi.com）。
- PO 识别为「尽力而为的草稿」，单价 / 数量 / 金额 / 品名务必逐项人工核对。
